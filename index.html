The aim of this series of workshops is to give you an introduction to computer programming.

The computer language chosen for this course is Python (Python 2.7 to be precise) and we will use the Jupyter notebooks as our development environment. The reasons behind these choices will be explained during the introductory lecture. Please visit the resources page for instructions on how to install Python and the Jupyter notebook on your computer, and for a list of available online references for the Python language.

By the end of these series of workshops you should be able to:
- read and understand the inner workings of simple Python programs.
- Create simple computer programs to acquire data and solve scientific problems.
- Use libraries such as NumPy, SciPy, SimPy and matplotlib in order to carry out more advanced calculations and to visualise data.
